# Password Generator 

Password Generator is a simple frontend web application that allows users to generate a password based on selected criteria.

Users can customize the password format by choosing options such as length and which types of characters to include.

---

## Features

- Generates passwords instantly
- Customizable settings:
  - Password length
  - Uppercase letters
  - Lowercase letters
  - Numbers
  - Special characters

## Tech Stack
- JavaScript
- HTML/CSS

## How to Run
1. Clone the repo
2. Open the project folder
3. Double-click index.html
